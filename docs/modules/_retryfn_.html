retryFn

  • retryFn<T>(func: () => Promise<T>, options?: { count?: number; onceCallback?: (index: number) => void; retryFence?: number }): Promise<T>
    • Defined in retryFn.ts:20

    重复执行目标func直到func 返回resolve,或者达到设定的尝试执行次数

    example
    retryFn(() => new Promise((resolve, reject) => {
    const value = Math.random();
    if (value > 0.9) {
        resolve(value);
    } else {
        reject(value);
    }
}));
